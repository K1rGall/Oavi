## Логическая фильтрация бинарных изображений (вариант 3)

**Исходные изображения:** `./input/*.png`  
**Результаты обработки:** `./output_variant3/`  
**Скрипт:** `./variant3_logical_filter.py`

---

## Цель работы
Реализовать фильтрацию монохромного изображения логическим методом и оценить изменения с помощью разностного изображения.

---

## Используемые методы

### 1. Перевод в полутон
Перед бинаризацией изображение переводится в яркостный канал:

`Y = 0.299R + 0.587G + 0.114B`

### 2. Бинаризация по Оцу
Порог выбирается автоматически по максимуму межклассовой дисперсии:

`sigma_b^2(t) = w0(t) * w1(t) * (mu0(t) - mu1(t))^2`

Полученный бинарный результат сохраняется как:

`<имя>_mono_input.png`

### 3. Логическая фильтрация (вариант 3)
Используется majority-фильтр `3x3`:

- рассматривается окно `3x3`;
- считается число белых пикселей;
- если сумма `>= 5`, центральный пиксель = `255`, иначе `0`.

Формально:

`B_out(x,y) = 1, если sum_3x3(B_in) >= 5, иначе 0`

Результат сохраняется как:

`<имя>_filtered_variant3.png`

### 4. Разностное изображение
Для оценки изменений строится XOR:

`D = B_in XOR B_out`

Результат сохраняется как:

`<имя>_xor_diff.png`

- автоматическая бинаризация по Оцу;

Результаты показывают, что majority-фильтр уменьшает мелкие шумовые артефакты и делает бинарные объекты более устойчивыми.
