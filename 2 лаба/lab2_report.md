## Обесцвечивание и бинаризация растровых изображений (вариант 3)

**Исходные изображения:** `./input/*.png`  
**Результаты обработки:** `./output/`, `./output_w15/`, `./output_w50/`  
**Скрипт:** `./lab2_variant3.py`

---

## Цель работы
Реализовать перевод полноцветного изображения в полутоновое и адаптивную бинаризацию с разными размерами окна.

---

## Используемые методы

### 1. Перевод RGB в полутоновое изображение
Яркость каждого пикселя вычисляется по формуле:

`Y = 0.299R + 0.587G + 0.114B`

Результат сохраняется в файл `*_gray.bmp`.

### 2. Адаптивная бинаризация (вариант 3)
Для каждого пикселя используется локальный порог:

`T(x,y) = mean_window(x,y) - offset`

`B(x,y) = 255, если I(x,y) > T(x,y), иначе 0`

Параметры эксперимента:
- `offset = 0`;
- размеры окна: `3x3`, `15x15`, `50x50`.

В реализации для ускорения вычислений применяется интегральное изображение.

## Вывод
- Увеличение окна сглаживает локальные колебания порога.
- Окно `15x15` обычно даёт компромисс между подавлением шумов и сохранением деталей.
- Окно `50x50` ведёт себя более глобально: мелкие детали чаще теряются, но крупные структуры выделяются стабильнее.
- Для неравномерного освещения размер окна существенно влияет на итоговую бинаризацию.
