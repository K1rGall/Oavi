# Отчёт по лабораторной работе №2
**Тема:** Обесцвечивание и бинаризация растровых изображений (вариант 3)  
**Исходные изображения:** `./input/*.png`  
**Результаты обработки:** `./output/`

## Цель работы
Реализовать:
1. перевод полноцветного изображения в полутоновое;
2. бинаризацию полутонового изображения адаптивным методом (вариант 3: локальное среднее, окно `3x3`).

## Исходные данные
Использованы полноцветные изображения в форматах PNG/BMP/JPG/JPEG.  
Обработка выполняется скриптом `lab2_variant3.py`.

## 1. Приведение изображения к полутоновому
**Метод:** для каждого пикселя рассчитывается яркость как взвешенная сумма каналов RGB.

Формула:

`Y = 0.299R + 0.587G + 0.114B`

Результат сохраняется в формате BMP: `*_gray.bmp`.

## 2. Бинаризация (вариант 3)
### 2.1. Метод
Используется адаптивный порог на основе локального среднего в окне `3x3`.

Для пикселя `(x,y)`:

`T(x,y) = mean_3x3(x,y) - offset`

`B(x,y) = 255, если I(x,y) > T(x,y), иначе 0`

где `offset` — параметр смещения порога (по умолчанию `0`).

### 2.2. Реализация
Для быстрого расчёта локальных средних используется интегральное изображение (prefix sums), что позволяет получать сумму по окну за `O(1)` на пиксель.

## 3. Результаты
Для каждого входного файла формируются:
- `*_gray.bmp` — полутоновое изображение;
- `*_bin_v3.png` — бинарное изображение;
- `*_compare.png` — коллаж `исходное | полутоновое | бинарное`.

## Вывод
В работе реализованы оба этапа обработки без библиотечных функций grayscale/binarization:
- ручное обесцвечивание;
- адаптивная бинаризация по локальному среднему в окне `3x3`.

Полученные результаты демонстрируют устойчивость локального порога к неравномерной освещённости по сравнению с глобальной бинаризацией.
