# Лабораторная работа 2 (вариант 3)

## Что делает скрипт

1. Переводит полноцветное RGB-изображение в полутоновое:
   `Y = 0.299R + 0.587G + 0.114B`.
2. Выполняет адаптивную бинаризацию по локальному среднему в окне `3x3`.

## Запуск

```bash
py "2 лаба/lab2_variant3.py"
```

По умолчанию вход берется из папки:

```text
2 лаба/input
```

Результаты сохраняются в:

```text
2 лаба/output
```
